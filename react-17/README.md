# React 17

React 17 is a **stepping-stone release**: few new developer-facing APIs, but important **tooling and event** changes that make upgrades safer.

## Features in this folder

- [New JSX transform](./new-jsx-transform.md) — optional compile output without `import React` in every file
- [Event delegation at the root](./event-delegation-root.md) — events attach to the React root, not `document`
