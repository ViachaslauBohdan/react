# React features by version (15–19)

Short, focused notes: **what problem a feature solves**, **how to use it**, and **tiny examples**. Each topic lives in its own file under the matching React major folder.

## How to read this

1. Open a version folder below (or the **Debugging** section for DevTools).
2. Use that folder’s `README.md` as a table of contents.
3. Open individual `.md` files for one feature at a time.
