# React 19

React 19 adds **Actions**, richer **form** hooks, the **`use` API**, simpler **refs**, **document metadata** from components, and the optional **React Compiler**.

## Features in this folder

- [use](./use-hook.md) — read a Promise or Context with Suspense-friendly behavior
- [useActionState](./use-action-state.md) — state updates driven by async actions (often forms)
- [useOptimistic](./use-optimistic.md) — optimistic UI while an action completes
- [useFormStatus](./use-form-status.md) — pending state from a parent `<form>`
- [ref as a normal prop](./ref-as-prop.md) — less `forwardRef` boilerplate for function components
- [Document metadata](./document-metadata.md) — `<title>`, `<meta>` colocated with routes/components
- [React Compiler](./react-compiler.md) — optional build-time memoization
