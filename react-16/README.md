# React 16

React 16 introduced **Fiber**, **error boundaries**, **portals**, and **fragments**—big steps toward resilient UIs and better composition.

## Features in this folder

- [Error boundaries](./error-boundaries.md) — catch render errors in the tree below
- [Portals](./portals.md) — render children into another DOM node (modals, tooltips)
- [Fragments](./fragments.md) — group elements without an extra DOM wrapper
- [Fiber (overview)](./fiber-overview.md) — what “Fiber” means for you as an app author
